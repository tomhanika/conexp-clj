context-descriptions

List of contexts and how they were obtained:

Context Source Description
Bird-Diet Found in lecture notes of Gerd Stumme. A context of birds and what they eat.
Brunson-Club Context created from the network found at http://konect.uni-koblenz.de/networks/brunson_club-membership. Membership information of corporate executive officers in social organisations.
Diagnosis Context scaled from the data found at https://archive.ics.uci.edu/ml/datasets/Acute+Inflammations. The data was created by a medical expert as a data set to test the expert system, which will perform the presumptive diagnosis of two diseases of the urinary system. The temperature attribute is interval-scaled.
Dolphins Context created from a network found at http://konect.uni-koblenz.de/networks/dolphins. A formal context created from a directed social network of bottlenose dolphins living in a fjord in New Zealand. A relation indicates frequent association based on observations between 1994 and 2001.
Forum-Romanum This context can be found in the standard work on FCA. A context based on ratings of monuments on the Forum Romanum in different travel guides and scaled ordinally.
Living-Beings-and-Water This context can be found in the standard work on FCA. The first formal context in the standard work on FCA (the yellow book) by Ganter and Wille.
Olympic-Disciplines Context taken from https://arxiv.org/abs/1908.08740 This context is about the disciplines of the Summer Olympic Games 2020.
Seasoning-Planner Found in lecture notes of Gerd Stumme. This context contains foods that are related to recommended seasonings based on a chart published by the spice company Fuchs Group.
Southern-Woman Context created from a network found at http://konect.uni-koblenz.de/networks/opsahl-southernwomen Participation of 18 white women in 14 social events over a nine-month period, collected in the Southern United States of America in the 1930s.
Wood-Properties Found in lecture notes of Gerd Stumme. A context about properties of different kinds of wood.
Cointoss-1 generated Artificially generated using coin-tossing.
Cointoss-2 generated Artificially generated using coin-tossing.
Dirichlet-1 generated Artificially generated using dirichlet distributions.
Dirichlet-2 generated Artificially generated using dirichlet distributions.
